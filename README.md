Descripción: 
ventas de productos alimenticios (distribución northwind)
NorthWind es una empresa distribuidora de productos alimenticios que vende a clientes (empresas y particulares) en múltiples países. 
El sistema debe soportar:
•	Gestión de clientes y su segmentación demográfica
•	Gestión de empleados vendedores y sus territorios
•	Catálogo de productos clasificados por categorías y proveedores
•	Procesamiento de pedidos con múltiples líneas de detalle
•	Envío de productos mediante transportistas externos
